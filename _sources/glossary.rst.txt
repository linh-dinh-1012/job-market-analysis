Glossaire
=========

API REST
  Interface HTTP exposant des ressources (JSON) via des endpoints.

Django REST Framework
  Outil pour construire une API REST sur Django (serializers, viewsets, permissions).

Reproductibilité
  Capacité à relancer les pipelines/analyses et obtenir les mêmes résultats (à données identiques).

Similarité sémantique
  Mesure de proximité entre textes (ici : intitulés de poste), souvent via vecteurs et cosine similarity.

Supabase
  Plateforme backend fournissant une base PostgreSQL managée, API REST, authentification, etc.  

Docker
  Outil de conteneurisation pour isoler et déployer des applications avec leurs dépendances.  

Render     
  Plateforme de déploiement cloud supportant les applications conteneurisées (Docker).

Streamlit
  Framework Python pour créer des applications web interactives de data science rapidement.
